# 📈 Using News to Predict Stock Movements

## 📌 Project Overview

This project explores how financial news can be used to predict stock market movements using Machine Learning and Natural Language Processing (NLP). The objective is to analyze the relationship between news sentiment and stock price fluctuations by combining historical market data with financial news articles.

The project demonstrates a complete data science workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model development, and evaluation.

---

## 🚀 Objectives

- Analyze financial news and stock market data
- Perform data cleaning and preprocessing
- Explore market trends using EDA
- Engineer meaningful features from news data
- Build Machine Learning models for stock movement prediction
- Evaluate model performance using suitable metrics

---

## 📊 Dataset

This project uses the **Two Sigma Financial News Dataset** available on Kaggle.

The dataset contains:

- Historical stock market data
- Financial news articles
- Asset information
- Market movement labels
